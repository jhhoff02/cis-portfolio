Fibonacci Series - The click of a button should call a function that calculates the numbers and shows it in the browser.


Seconds Calculation

  • The first and second textboxes take inputs as number of hours and minutes.
  • When the button is clicked, a function should be called to calculate the number of seconds based on hours and minutes provided.
  • The third textbox should not have any border. Use style="border:none" in the input statement. When the button is clicked, the function should set the value of the third textbox as the number of seconds.
